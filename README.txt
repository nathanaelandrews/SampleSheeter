Sample Sheeter is designed to generate sample sheets for the
Enge Lab, for sending single cell sequencing libraries to BEA 
or Novogene.

To run Sample Sheeter:

1. Run the command: python getSampleSheet.py
2. Follow the prompts.

The final sequencing sheet(s) will be saved in the sampleSheets 
folder. Sample sheets for individual libraries (plates) will 
be saved in hidden subfolders of the sampleSheets folder.
